# Documentation

This directory preserves supporting materials from the Knowledge Graph RAG Assistant senior design capstone project, originally developed under the ACME10-HE-RAGApp name.

## Contents

- `project-report/` - final report, abstract, traceability matrix, prototype report, and final client presentation.
- `sprint-reports/` - sprint reports and sprint presentation decks.
- `meeting-notes/` - client and team meeting notes.
- `code-guides/` - notebooks used to explore knowledge graphs, BERT embeddings, and FAISS vector search.
- `performance-stats.md` - retrieval and runtime notes collected during development.
- `images/` - project and team images.

For setup and usage instructions, see the root [README](../README.md).
